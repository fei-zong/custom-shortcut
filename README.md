A custom site navigation plugin that allows users to add and manage website shortcuts with logos.

== Description ==

Custom Site Navigator allows users to create a personalized dashboard of website shortcuts. Users can add sites with names, URLs, and logos, and arrange them in their preferred order.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-site-navigator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcode `[custom_site_navigator]` to display the navigator on any page or post.
